# URL
- Uniform Resource Locator
- 인터넷의 특정 Resource의 정확한 위치 정보(주소)를 나타내는 문자열


# URN
- 자원의 이름만을 나타내는 문자열


# URI
- Uniform Resource Identifier
- URI는 웹 페이지, 파일 또는 데이터베이스 항목과 같은 리소스를 식별하거나 이름을 지정하는 문자열
- URL보다 더 일반적인 용어로 URL을 포함합니다. URI는 리소스의 위치나 인터넷 액세스 여부와는 관련이 없습니다
- 특정 resource의 이름 또는 위치를 나타내는 총칭
